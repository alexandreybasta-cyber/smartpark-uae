QUESTIONS

Is SpotSense live today?

No. SpotSense is a concept built for the Qwen Cloud Hackathon. The JBR pilot above is the proposal on the table.

Why no cameras?

mmWave radar reads presence, not identity. No plates, no faces, nothing stored that can leak. Enforcement stays with Parkin's own systems.

What does one bay cost?

Under forty dirhams in parts — LD2410, ESP32-S3, enclosure. A 0.4 W draw runs on solar trickle. Install is curb-flush, one visit.

What do the agents actually decide?

Qwen-Max arbitrates enforcement — grace, settle, or dispatch. Qwen-Plus whispers open bays into drivers' navigation. Shared memory keeps both honest about what each bay tends to do.

SpotSense is a concept. All metrics are projections.
